# PKI & TOTP Microservice

This repository contains the secure, containerized microservice that demonstrates enterprise-grade security practices through PKI and TOTP two-factor authentication.

## Setup Instructions

**1. Create virtual environment & install requirements (Optional but recommended)**
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

**2. Generate Student Key Pair**
Run the key generation script. This creates `student_private.pem` and `student_public.pem`.
```bash
python scripts/generate_keys.py
```

**3. Download Instructor Public Key**
Download the instructor's public key from your course resources and save it as `instructor_public.pem` in the root of this folder.

**4. Request Encrypted Seed**
Call the instructor API to receive your encrypted seed. Replace the placeholders with your actual details:
```bash
python scripts/request_seed.py --student-id "YOUR_STUDENT_ID" --repo-url "https://github.com/yourusername/your-repo-name"
```
*Note: Make sure your repository is exactly the URL you will submit!*

**5. Start the Microservice (Docker)**
Build and run the application using Docker Compose:
```bash
docker-compose up --build -d
```

## Testing the API

**Decrypt Seed:**
```bash
# PowerShell
$seed = Get-Content -Raw -Path encrypted_seed.txt
Invoke-RestMethod -Uri "http://localhost:8080/decrypt-seed" -Method Post -Body (@{encrypted_seed=$seed} | ConvertTo-Json) -ContentType "application/json"
```

**Generate 2FA Code:**
```bash
Invoke-RestMethod -Uri "http://localhost:8080/generate-2fa" -Method Get
```

**Verify 2FA Code:**
```bash
Invoke-RestMethod -Uri "http://localhost:8080/verify-2fa" -Method Post -Body '{"code": "123456"}' -ContentType "application/json"
```

**Check Cron Job Logs:**
Wait at least 70 seconds, then view the logs generated by the background cron job:
```bash
docker exec pki-2fa-microservice cat /cron/last_code.txt
```

   git add .
   git commit -m "Complete microservice implementation"

   ```bash

   ```
